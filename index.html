ACT1 PEC2 – Gráfico de Área

Definición y funcionamiento del gráfico

El gráfico de área es una técnica de visualización estadística basada en el gráfico de líneas. Se utiliza para representar la evolución de datos cuantitativos a lo largo del tiempo. La diferencia principal radica en que el área entre la línea y el eje horizontal se rellena con color, lo cual ayuda a destacar el volumen y la relación entre distintas series.

Su origen se encuentra en los estudios económicos y demográficos del siglo XX. Es comúnmente aplicado en informes financieros, análisis de tendencias de mercado, evolución demográfica, energía, clima, tráfico web y muchas otras disciplinas.

Análisis del dataset

  • Fuente: Epdata
  • Tipo: Series temporales cuantitativas.
  • Variables: Año, país (China/EE.UU.), exportaciones, importaciones.
  • Estructura: Dataset limpio con datos por país y año (formato largo).
  • Limitaciones: La comparación entre múltiples países es efectiva visualmente hasta cierto límite (demasiadas series saturan la gráfica).

Visualización realizada

Se representó la evolución de las exportaciones e importaciones de España con China y Estados Unidos desde 1995 hasta 2023. Se utilizaron colores degradados emparejados para reflejar la vinculación entre exportación e importación de cada país.

Objetivo y análisis

Esta visualización permite comparar cómo ha evolucionado el comercio exterior bilateral a lo largo del tiempo. El uso del área y color facilita la percepción del crecimiento, estancamiento o cambios abruptos. Destaca el crecimiento progresivo con China y la irregularidad con EE.UU. Esta técnica es útil para comunicar tanto magnitudes absolutas como tendencias comparadas.

ACT2 PEC2 – Matriz de Correlación

Definición y funcionamiento del gráfico

Una matriz de correlación es una representación cuadrada que muestra la relación lineal entre pares de variables numéricas. Se acompaña frecuentemente de un mapa de calor (heatmap) para una interpretación visual rápida. Los valores van de -1 (correlación negativa perfecta) a +1 (positiva perfecta), siendo 0 la ausencia de correlación.

Esta técnica se usa ampliamente en análisis exploratorio de datos (EDA), estadística y machine learning como paso previo a la selección de variables o detección de colinealidades.

Análisis del dataset

  • Fuente: UCI Machine Learning Repository
  • Tipo: Cuantitativo, multivariable.
  • Variables: Edad, peso, distancia al trabajo, IMC, número de hijos, carga laboral, etc.
  • Observaciones: 740 registros de empleados.
  • Limitaciones: No se incluyen variables emocionales o cualitativas que podrían incidir en el ausentismo.

Visualización realizada

Se aplicó un heatmap para visualizar la correlación entre variables del conjunto de datos de ausentismo. Se destacan relaciones esperadas y la ausencia de relación fuerte entre variables y el tiempo de ausencia.

Objetivo y análisis

El objetivo era explorar posibles predictores del ausentismo laboral. A pesar de la falta de correlaciones fuertes con el target (ausentismo), se identificaron relaciones útiles entre variables, como edad y tiempo de servicio, o peso e IMC. Esto sugiere que el fenómeno requiere análisis multicausal.

ACT3 PEC2 – Gráfico OHLC

Definición y funcionamiento del gráfico

El gráfico OHLC (Open-High-Low-Close) se utiliza en análisis técnico para representar cómo se comporta el precio de un activo en un periodo (habitualmente un día). Cada unidad muestra cuatro valores: apertura, máximo, mínimo y cierre. Se puede representar como velas japonesas o líneas independientes.

Es ampliamente usado en bolsas de valores y plataformas financieras. Proporciona información crítica para la toma de decisiones de inversión.

Análisis del dataset

  • Fuente: Investing.com
  • Tipo: Serie temporal de datos bursátiles.
  • Variables: Fecha, Open, High, Low, Close.
  • Periodicidad: Diaria.
  • Limitaciones: No incluye datos cualitativos (noticias, eventos económicos) ni volumen de transacciones.

Visualización realizada

Dado que no se pudo generar un gráfico de velas tradicional en el entorno, se utilizó una versión por líneas que representa cada valor (OHLC) de forma independiente a lo largo del tiempo para el índice IBEX 35.

Objetivo y análisis

El gráfico permite observar la evolución de precios, así como la volatilidad del mercado. Es útil para identificar días de alta variabilidad, tendencias a corto plazo y posibles señales de cambio en el comportamiento del índice.
